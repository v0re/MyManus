# 股票分析技能

> 本技能提供全面的股票與公司分析能力，整合多種金融市場資料來源。

## 核心能力

- **公司基本資料**：獲取公司概況、產業、市值、員工數等基本資訊。
- **股票技術分析**：計算和分析各種技術指標，如移動平均線、相對強弱指數 (RSI) 等。
- **價格圖表生成**：生成歷史股價圖表，並可疊加技術指標。
- **內部持股分析**：檢索公司內部人士（如高階主管、大股東）的持股變動情況。
- **SEC 文件檢索**：搜尋並下載美國證券交易委員會 (SEC) 的公開文件，如 10-K（年報）、10-Q（季報）等。
- **財務指標分析**：分析公司的財務報表，計算如每股盈餘 (EPS)、本益比 (P/E) 等關鍵財務指標。

## 使用場景

- **投資研究**：在投資前對目標公司進行深入的基本面和技術面分析。
- **市場監控**：追蹤特定股票或市場板塊的動態。
- **學術研究**：在金融或經濟學研究中，獲取和分析歷史市場資料。
- **競爭對手分析**：分析上市競爭對手的財務狀況和市場表現。

## 工作流程

1.  **定義分析目標**：明確要分析的公司、股票代碼以及分析的重點（例如，是想看基本面、技術面，還是最新的 SEC 文件）。
2.  **收集資料**：根據分析目標，使用相關工具或 API 獲取所需資料。
3.  **分析與處理**：對收集到的原始資料進行處理和分析，例如計算技術指標、整理財務數據等。
4.  **視覺化呈現**：將分析結果以圖表或表格的形式呈現，使其更直觀易懂。
5.  **生成報告**：將所有分析結果和視覺化圖表整合到一份完整的分析報告中。

## 範例：分析蘋果公司 (AAPL)

**使用者請求**：
> 「請幫我全面分析一下蘋果公司 (AAPL) 的股票，我需要它的基本資料、近一年的股價圖、最新的財務狀況以及高層的持股變動。」

**AI 代理執行流程**：

1.  **[INFO]** 收到請求，開始分析蘋果公司 (AAPL)。
2.  **[執行中]** 獲取公司基本資料：
    -   使用 `yfinance` 或類似工具獲取公司概況。
3.  **[執行中]** 生成股價圖表：
    -   獲取近一年歷史股價資料。
    -   使用 `matplotlib` 或 `plotly` 繪製股價圖，並疊加 50 日和 200 日移動平均線。
    -   將圖表儲存為 `AAPL_price_chart.png`。
4.  **[執行中]** 分析最新財務狀況：
    -   檢索最新的 10-K 或 10-Q 文件。
    -   提取關鍵財務數據，如營收、淨利、EPS 等。
    -   整理成表格。
5.  **[執行中]** 分析內部持股變動：
    -   從相關財經資料網站或 API 檢索內部交易記錄。
    -   篩選出近半年的重要交易。
6.  **[RESULT]** 蘋果公司 (AAPL) 分析報告已完成。報告包含公司基本資料、近一年股價圖、最新財務摘要及內部持股變動。詳情請見附件。
    -   附件：`AAPL_analysis_report.md`, `AAPL_price_chart.png`

## 所需工具與資料來源

- **Python 套件**：`yfinance`, `pandas`, `numpy`, `matplotlib`, `sec-api`
- **資料來源**：Yahoo Finance, SEC EDGAR 数据库, 各大財經新聞網站 API。

## 注意事項

-   金融市場資料可能存在延遲，請注意資料的即時性。
-   所有分析結果僅供參考，不構成任何投資建議。
-   使用外部 API 或服務時，請注意其使用條款與頻率限制。
