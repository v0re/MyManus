# 網路技能搜尋器

> 本技能專門用於在網路上（特別是 GitHub）搜尋、評估和推薦高品質的 AI 代理技能 (Agent Skills) 或相關外掛程式。

## 核心能力

- **技能搜尋**：使用精準的關鍵字在 GitHub 等平台上搜尋符合使用者需求的技能。
- **品質評估**：根據星級、更新頻率、文件完整性和社群活躍度等指標，評估開源專案的品質。
- **相容性分析**：分析技能的技術堆疊，判斷其與當前代理環境的相容性。
- **推薦與比較**：提供多個候選技能的比較，並根據使用者需求推薦最合適的選項。
- **安裝指引**：提供清晰的步驟，指導使用者如何安裝和設定所選的技能。

## 使用場景

- **擴展代理能力**：當使用者希望 AI 代理具備某項新能力時，使用此技能尋找現成的解決方案。
- **探索新工具**：使用者想了解特定領域（如資料視覺化、社群媒體管理）有哪些可用的 AI 技能。
- **尋找替代方案**：當現有技能無法滿足需求時，尋找功能更強大或更適合的替代品。

## 工作流程

1.  **理解需求**：與使用者溝通，明確所需技能的功能、應用場景和技術偏好。
2.  **制定搜尋策略**：組合關鍵字，如 `agent skill`、`claude code plugin`、`langchain tool` 搭配具體的功能描述（例如 `video editing`、`pdf generation`）。
3.  **執行搜尋**：在 GitHub 上進行搜尋，並篩選出初步的候選清單。
4.  **評估與篩選**：
    -   **星級 (Stars)**：通常 100+ 表示有一定關注度，1k+ 表示相當可靠。
    -   **最後提交 (Last commit)**：偏好半年內有更新的專案，避免選擇已停止維護的專案。
    -   **文件 (Documentation)**：檢查 `README.md` 是否清晰，是否有提供使用範例。
    -   **Issues**：瀏覽開放和已關閉的 Issues，了解專案的活躍度和問題處理情況。
5.  **提供推薦**：將排名前 2-3 的技能整理成表格，比較其優缺點，並給出最終推薦。
6.  **提供安裝指南**：為推薦的技能提供詳細的安裝和設定步驟。

## 範例：尋找一個用於簡報製作的技能

**使用者請求**：
> 「我想讓我的 AI 代理能夠製作 PPT 簡報，有什麼推薦的技能嗎？」

**AI 代理執行流程**：

1.  **[INFO]** 收到請求，開始搜尋簡報製作相關的 Agent Skills。
2.  **[執行中]** 制定搜尋關鍵字：「agent skill presentation」、「claude code plugin pptx」、「langchain tool slides」。
3.  **[執行中]** 在 GitHub 上搜尋，初步篩選出 `MistoAI/Misto`、`ai-modules/slides-generator` 等幾個專案。
4.  **[執行中]** 評估各專案的星級、更新頻率和文件品質。
5.  **[ASK]** 我找到了幾個選項，您對簡報的視覺風格有什麼偏好嗎？例如，您喜歡傳統的 HTML/CSS 樣式，還是更具設計感的圖片式投影片？
6.  **[INFO]** 根據您的偏好，我推薦 `MistoAI/Misto`，它支援多種輸出格式且社群活躍。以下是它的基本資訊和安裝方式。
7.  **[RESULT]** 已為您找到合適的簡報製作技能。詳細的比較和安裝指南請見附件。
    -   附件：`presentation_skills_comparison.md`

## 注意事項

-   開源專案的品質參差不齊，務必進行仔細評估，避免推薦不穩定或已廢棄的專案。
-   優先推薦與當前環境（如 Claude Code, LangChain）有良好整合的技能。
-   在推薦時，應清楚說明技能的授權協議 (License)，提醒使用者注意合規性。
